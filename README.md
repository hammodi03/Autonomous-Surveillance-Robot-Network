# Autonomous-Surveillance-Robot-Network

GuardRobot is an autonomous indoor security robot built using an ESP32/Arduino-based embedded system, together with a Python server for gathering data.
The robot navigates between rooms with the help of IR sensors, detects people and objects with the use of radar, and manages its own battery level by returning to a charging station when needed.
